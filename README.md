# Wordle

This is a clone of the wordle, recreate with SwiftUI

## Requiremnets

1. You need a computer with xcode 
2. You need to install [cocoapods](https://cocoapods.org)


```bash
sudo gem install cocoapods
```

## Usage

1. Install the pods

```bash
pod install 
```
